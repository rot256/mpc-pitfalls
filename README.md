# Common MPC Pitfalls

A collection of common mistakes when implementing multi-party computation protocols,
intended to help developers implement MPC securely.

## Layout

- `content/pitfalls/mini/` -- the pitfall entries, one Markdown file each.
- `content/pitfalls/mini/classes/` -- the category introductions; each pitfall is grouped under a class.
- `layouts/index.html` -- the single-page template that renders all entries.
- `old/` -- the legacy, free-form pitfall notes that predate the structured entries. Kept for reference; not built.
